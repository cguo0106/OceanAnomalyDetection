# Code for Resilient Anomaly Detection in Ocean Drifters with Unsupervised Learning, Deep Learning Models, and Energy-Efficient Recovery

This repository contains the code used for experiments in the paper.

Data was taken from the Global Drifter Program - 6 Hour Interpolated QC Drifter Data from NOAA Atlantic Oceanographic and Meteorological Laboratory. 
It can be found on this link: https://erddap.aoml.noaa.gov/gdp/erddap/tabledap/drifter_6hour_qc.html.
